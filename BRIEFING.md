# Content Builder 사용 가이드

작업자를 위한 Content Builder 사용 방법 안내입니다.

---

## 1. 시작하기

### 프로그램 실행

```bash
# 프로젝트 폴더로 이동
cd content-builder

# 개발 서버 실행
npm run dev
```

브라우저에서 **http://localhost:5173/** 접속

---

## 2. 기존 콘텐츠 수정하기

### Step 1: 폴더 Import

1. 상단의 **"Import Folder"** 버튼 클릭
2. 파일 선택 창에서 **과목코드 폴더** 선택
   - 예: `subjects/25itinse/` 폴더 선택
3. 확인 버튼 클릭

**성공 시 알림:**
```
30개 차시를 성공적으로 불러왔습니다!

과목코드: 25itinse
과정명: 인터넷보안
이미지: 45개 저장됨
```

### Step 2: 내용 수정

- **왼쪽 사이드바**: 차시 선택
- **중앙 영역**: 내용 편집
- **오른쪽 사이드바**: 과목/교수 정보

### Step 3: Export

1. **"Export to Subjects"** 버튼 클릭
2. 출력 경로 입력 (기본: `~/Documents`)
3. JSON 파일이 다운로드됨
4. **터미널에서 실행:**

```bash
python3 builder_to_subjects.py ~/Downloads/25itinse_builder.json ~/Documents
```

5. `~/Documents/25itinse/` 폴더에 결과물 생성됨

---

## 3. 새 콘텐츠 만들기

### Step 1: 과목 정보 입력

오른쪽 사이드바에서:
- **과목 코드**: `25itinse` 형식
- **과정명**: `인터넷보안` 등

### Step 2: 교수 정보 입력

오른쪽 사이드바에서:
- **이름**: 교수명
- **사진 URL**: 교수 사진 경로
- **학력**: 줄바꿈으로 구분
- **경력**: "추가" 버튼으로 항목 추가, 시작일/종료일 선택

### Step 3: 차시 추가

1. 왼쪽 사이드바에서 **"+ 새 차시"** 클릭
2. 기본 정보 입력:
   - 차시명
   - 주차 번호

### Step 4: 섹션별 내용 입력

각 섹션을 순서대로 입력:

| 섹션 | 입력 항목 |
|------|----------|
| 준비하기 | 오리엔테이션(1주1차시만), 용어체크, 학습목표 |
| 학습하기 | 생각묻기, 강의 URL/자막, 점검하기 |
| 정리하기 | 연습문제, 학습정리, 다운로드 URL |

## 4. 에디터 사용법

### 리치 텍스트 에디터

용어 내용, 교수님 의견, 연습문제, 학습정리 필드에서 사용 가능

**툴바 기능:**
| 버튼 | 기능 | 단축키 |
|------|------|--------|
| H1, H2, H3 | 제목 | `# `, `## `, `### ` + 스페이스 |
| **B** | 굵게 | Ctrl+B |
| *I* | 기울임 | Ctrl+I |
| U | 밑줄 | Ctrl+U |
| ~~S~~ | 취소선 | - |
| • | 글머리 기호 | `- ` + 스페이스 |
| 1. | 번호 목록 | `1. ` + 스페이스 |
| " | 인용 | `> ` + 스페이스 |
| </> | 코드 블록 | ``` + Enter |
| 이미지 | 이미지 삽입 | - |
| 표 | 표 삽입 | - |

### 이미지 삽입 방법

1. **버튼 클릭**: 툴바의 "이미지" 버튼 → 파일 선택
2. **드래그 앤 드롭**: 이미지 파일을 에디터로 끌어다 놓기
3. **붙여넣기**: 이미지 복사 후 Ctrl+V

### Import된 이미지 표시

- **파란 점선 테두리**: 원본 이미지 경로가 있는 이미지
- **마우스 호버**: 원본 경로 확인 가능
- Export 시 자동으로 이미지 파일 복원됨

## 5. 연습문제 작성

### 문제 유형

| 유형 | 설명 |
|------|------|
| OX | O/X 선택 문제 |
| 4지선다 | 4개 선택지 중 1개 선택 |

### 작성 방법

1. **"+ 문제 추가"** 버튼으로 문제 추가
2. 문제 유형 선택 (OX / 4지선다)
3. **문항**: 문제 내용 입력 (이미지 삽입 가능)
4. **선택지**: 4지선다인 경우 4개 입력
5. **정답**: 정답 선택
6. **해설**: 정답 해설 입력 (이미지 삽입 가능)

---

## 6. 주의사항

### Import 시
- 반드시 **과목코드 폴더**를 선택해야 합니다
- 예: `25itinse/` (O), `subjects/` (X), `01/` (X)

### Export 시
- JSON 파일 다운로드 후 **반드시 Python 스크립트 실행** 필요
- Python 3가 설치되어 있어야 합니다

### 이미지 관련
- Import된 이미지는 메모리에 저장됨 (브라우저 새로고침 시 사라짐)
- 작업 중간에 Export하여 백업 권장

---

## 7. 문제 해결

### "data.json 파일을 찾을 수 없습니다"
→ 올바른 과목코드 폴더를 선택했는지 확인

### Export 후 이미지가 없음
→ Python 스크립트 실행 여부 확인
```bash
python3 builder_to_subjects.py <json파일> <출력경로>
```

### 에디터가 느림
→ 이미지가 많은 경우 발생 가능. 저장 후 새로고침 권장
