## 상세 요구사항

### 1. 임포트/익스포트

#### 1.1 H1/불릿 서식 유지
- **현상**: 임포트 후 H1, 불릿(체크 포함) 등이 해제되어 보임
- **실제**: 실서버에서는 정상 표시되지만 에디터에서 해제된 것처럼 보임
- **요청**: 에디터에서도 원본 서식 그대로 보이게 해야 함

#### 1.2 임포트 후 검수 기능
- **현상**: 임포트 후 오류 확인을 위해 모든 페이지를 일일이 확인해야 함
- **요청**:
  - 텍스트 형식 불일치 자동 감지
  - 누락된 필드 자동 검토
  - 주차/차시별 오류 리포트

#### 1.3 마지막 줄바꿈 표시
- **현상**: 복사-붙여넣기 시 마지막에 불필요한 엔터가 포함되어 여백 발생
- **요청**:
  - ~~삭제~~ → 삭제하지 않고 **표시**만 (시각적 확인용)
  - 한글 공백 기호(¶)처럼 눈에 보이게

#### 1.4 교강사 경력 파싱 버그
- **현상**: 기간(연도)이 내용 란에 함께 들어감
- **예시**: "2020년 1월 ~ 2023년 12월<br>OO대학 교수" → 기간과 내용 분리 안 됨
- **요청**: 기간은 기간 필드에, 내용은 내용 필드에 정확히 분리

---

### 2. 이미지

#### 2.1 이미지 크기 편집
- **현상**: 원본 크기 그대로 삽입되어 크기가 중구난방
- **요청**:
  - 업로드 시 크기 조절 팝업 또는
  - 에디터 내에서 드래그로 크기 조절 가능하게
  - 적정 크기로 자동 맞춤 옵션 (권장 크기 안내)

#### 2.2 실서버와 동일한 표시
- **현상**: 에디터에서 보이는 크기와 실서버에서 보이는 크기가 다름
- **요청**: 에디터에서도 실서버와 동일하게 렌더링

---

### 3. 표(Table)

#### 3.1 행간/패딩 축소
- **현상**: 표 셀의 행간과 패딩이 너무 커서 스크롤이 길어짐
- **요청**: 더 컴팩트한 표 스타일 (한글 문서 표 수준)

#### 3.2 표 크기 조정
- **현상**: 열 너비를 조정할 수 없음
- **요청**:
  - 열 너비 드래그 조정
  - "열 너비 같게" 기능

#### 3.3 표 익스포트 방식
- **현상**: 표가 이미지로 변환되어 익스포트됨
- **문제**: 나중에 수정 불가능
- **요청**: HTML 테이블 그대로 유지 옵션 (이미지 변환 선택)

---

### 4. 에디터 기능

#### 4.1 용어 체크 - 타이틀 줄바꿈
- **현상**: 화면에서 몇 글자까지 한 줄인지 가늠 안 됨
- **요청**:
  - 실서버 렌더링 폭과 동일한 텍스트 박스 제공
  - 또는 미리보기에서 실제 줄바꿈 위치 확인

#### 4.2 용어 체크 - 내용 자동 분리
- **현상**: 한 번에 여러 줄 붙여넣으면 수동으로 분리해야 함
- **요청**:
  - 엔터 기준으로 자동 항목 분리
  - 또는 불릿 기준 분리

#### 4.3 문항/문제 색상 구분
- **현상**: 문항 입력 칸과 선택지 입력 칸이 비슷해서 실수로 잘못 입력
- **요청**: 문제 영역과 선택지 영역 배경색 구분

---

### 5. 타임스탬프

#### 5.1 학습내용과 자동 연동
- **현상**: 학습내용 3개면 타임스탬프도 3개 필요한데 수동 추가
- **요청**:
  - 학습내용 개수 변경 시 타임스탬프 자동 추가/삭제
  - 기본 세팅으로 개수 맞춤 (값은 사용자가 입력)
  - 실습 과목도 동일 적용

---

### 6. 네비게이션

#### 6.1 URL 차시 변경 시 리다이렉트 방지
- **현상**: URL에서 차시 번호만 변경하면 첫 페이지로 이동
- **예시**: `/course/01/page/5` → `/course/02/page/5` 입력 시 → 02 차시 1페이지로 이동
- **과거**: 예전에는 해당 페이지 그대로 유지됨
- **요청**: 차시 변경 시 페이지 번호 유지

---

### 7. 외부 연동

#### 7.1 구글 닥스 강의계획서 연동
- **현상**: 강의계획서가 구글 닥스로 관리됨
- **요청**:
  - 구글 닥스에서 차시명, 학습내용, 학습목표 자동 불러오기
  - 최소한 차시명이라도 연동
  - 일괄 넣기 기능에 활용
